# TasteBite Frontend

## Описание
Frontend приложения для системы управления рестораном TasteBite на React + Vite + Tailwind CSS.

## Технологии
- React 18
- Vite
- React Router v6
- Tailwind CSS
- Axios
- React Hot Toast
- React Icons
- date-fns
- Recharts

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение в режиме разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Структура проекта

```
src/
├── components/      # Переиспользуемые компоненты
├── contexts/       # React Context для состояния
├── pages/          # Страницы приложения
│   ├── client/     # Страницы клиента
│   ├── staff/      # Страницы сотрудника
│   └── admin/      # Страницы администратора
├── services/       # API сервисы
└── utils/          # Утилиты

```

## Основные функции

### Для всех пользователей:
- Просмотр меню с категориями
- Добавление блюд в корзину
- Оформление заказа
- Авторизация и регистрация

### Для клиентов:
- Личный кабинет с историей заказов
- Бронирование столиков
- Управление профилем
- Бонусная программа

### Для сотрудников:
- Управление заказами
- Управление бронированиями
- Изменение доступности блюд
- Просмотр статистики

### Для администраторов:
- Полное управление меню
- Управление пользователями
- Просмотр аналитики и отчетов
- Настройки системы

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`.
